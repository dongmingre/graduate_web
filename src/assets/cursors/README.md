# Cursor Assets

The chalk cursor PNGs are derived from the CC0 PurePNG image "Color Chalk Piece":

https://purepng.com/photo/2941/color-chalk-piece

The source image was cropped to the yellow-green chalk stick, cleaned to the largest alpha component, remapped to a warm white chalk tone, rotated into a natural pointer angle, enhanced for 64px cursor readability, and exported as:

- `chalk.png`
- `chalk-press.png`
